# Quickstart Guide

This guide provides a brief overview of how to set up the development environment for the "Physical AI & Humanoid Robotics" book.

## 1. Prerequisites

- Ubuntu 22.04
- Python 3.10
- Git

## 2. ROS 2 Installation

Follow the official ROS 2 Humble installation guide:
[https://docs.ros.org/en/humble/Installation.html](https://docs.ros.org/en/humble/Installation.html)

## 3. Gazebo Installation

Follow the official Gazebo Fortress installation guide:
[https://gazebosim.org/docs/fortress/install](https://gazebosim.org/docs/fortress/install)

## 4. NVIDIA Isaac Sim Installation

Follow the official NVIDIA Isaac Sim installation guide:
[https://docs.omniverse.nvidia.com/isaacsim/latest/installation/install_workstation.html](https://docs.omniverse.nvidia.com/isaacsim/latest/installation/install_workstation.html)

## 5. Unity Installation

Install Unity Hub and the 2022.3 LTS version of the Unity Editor.

## 6. OpenAI API Key

You will need an OpenAI API key to run the LLM examples. You can get one from the OpenAI website.
